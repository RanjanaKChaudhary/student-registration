# Student-Course Registration App

## Description
This is a simple **Node.js and Express** web application for registering students for courses. Users can submit their **name, email, and course** through a registration form. All registered students are displayed in a dynamic, responsive table. The app uses **MongoDB** to store student information.

This project is intended for **practice and learning purposes**, demonstrating:
- CRUD operations with **MongoDB and Mongoose**
- Server-side rendering with **EJS**
- Routing in **Express.js**
- Basic form handling and validation
- Environment variable management with `.env`
- Clean UI styling with **Bootstrap 5**

---

## Features
- Register students with **name, email, and course**
- View a list of **all registered students**
- Responsive and attractive **Bootstrap-styled UI**
- Use of **.env file** for database and port configuration
- Easy to extend for authentication or additional functionality

---

## Technologies Used
- **Node.js** – JavaScript runtime
- **Express.js** – Web framework
- **MongoDB** – Database to store student information
- **Mongoose** – MongoDB object modeling
- **EJS** – Template engine for server-side rendering
- **Bootstrap 5** – Styling and responsive UI
- **dotenv** – Environment variable management
